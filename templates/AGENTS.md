# Research Agent Start Protocol

Before doing any work:

1. Read `_research_context/START_HERE.md`.
2. Read `RESEARCH_HUB.md` when present.
3. Read `_research_context/latest_research_state.md`.
4. Read `_research_context/latest_claim_boundaries.md`.
5. Search `_research_context/document_chunks.jsonl` before scanning raw
   folders.
6. For document aggregation, related work, reports, or cross-workspace
   comparison, use Research Hub context before opening raw files.
7. Treat `_research_context/` as generated read-only context.
8. Do not modify hub mirrors directly.
9. When reporting scores or claims, include scope, method, metric type, and
   evidence path when available.
