# Research Agent Start Protocol

Before doing any work:

1. Read `_research_context/START_HERE.md`.
2. Read `_research_context/latest_research_state.md`.
3. Read `_research_context/latest_claim_boundaries.md`.
4. Search `_research_context/document_chunks.jsonl` before scanning raw
   folders.
5. Treat `_research_context/` as generated read-only context.
6. Do not modify hub mirrors directly.
7. When reporting scores or claims, include scope, method, metric type, and
   evidence path when available.
